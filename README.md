## 📖 About This Repository

Welcome to the official **QCSSTUDIO Network and Network Security** repository — maintained by **Ravi Kant Sankhyan**, a Network & Network Security IT Consultant with **15+ years of hands-on experience**.

This repository serves as a central hub for network automation scripts, firewall configurations, penetration testing tools, cloud networking templates, lab topologies, and career resources for students and professionals in the Networking & Cybersecurity domain.

---

## 📁 What's Inside

| Folder | Description |
|--------|-------------|
| `automation/` | Python scripts for auto-configuring Cisco/Juniper devices using Netmiko & NAPALM |
| `firewall/` | Firewall rule audit scripts & hardened config templates for major vendors |
| `labs/` | GNS3 & EVE-NG lab files for CCNA/CCNP practice with full topology docs |
| `pentest/` | Network pen testing toolkit — scanning, enumeration & vulnerability assessment |
| `playbooks/` | Incident response playbooks, DDoS mitigation & network hardening checklists |
| `cloud/` | AWS & Azure network security templates with NACLs, VPNs & Terraform configs |
